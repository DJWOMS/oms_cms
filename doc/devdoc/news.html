3. Новости/Блог¶

Это модуль для ведение блога

3.1. Вывод списка статей¶

{% for post in post_list %}
    {% if post.image %}
        <img src="{{ post.image.url }}">
    {% endif %}
    <p class="">{{ post.published_date }}</p>
    <p class="">
        {{ post.title }}
    </p>
{% endfor %}
